# Vessel Project

This is a data engineering solution that reads flat files, transforms the data, uploads into an SQL database and visualizes the resulting data.

# TOOLS USED
1. Azure storage
2. Azure databricks
3. Azure data factory
4. Azure SQL server and database
5. Power BI
